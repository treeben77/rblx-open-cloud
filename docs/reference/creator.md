# Creator

Creators are a base class inherited by [`User`][rblxopencloud.User] and [`Group`][rblxopencloud.Group]. This reference documents APIs related to assets and the creator store.

Creator store endpoints to search the toolbox and asset saving are in [`ApiKey`][rblxopencloud.ApiKey] instead.
